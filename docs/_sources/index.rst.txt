Assignment_2
*********************

This is the documenation for the second assignment of the Experimental Robotics Laboratory course.
By clicking on 'src' the user will see the documentation of all the C++ scripts used to implement the assignment.
By clicking on 'scripts' the user will see the documentation of all the Python scripts used to implement the assignment.
By clicking on 'utilities', the user will see the documentation of a script used to define the name of each node, server, topic, and some useful parameter.
